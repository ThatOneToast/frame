# Typography

Text declarations express type role and emphasis.

```frame
text PageTitle {
  size heading
  weight bold
  color bright
}

text MutedLabel {
  size caption
  color muted
}

text CodeText {
  font mono
  size body
}
```

Use `text` or `color` values for semantic color intent, and `font`, `size`, and `weight` for type intent.
