# Surfaces

Surfaces describe visual layers with named intent.

```frame
card PanelCard {
  surface panel
  padding medium
}

card GlassCard {
  surface glass
  border soft
  shadow soft
}

card HeroCard {
  surface gradient midnight
  text bright
}
```

Semantic color intent:

```frame
button DeleteButton {
  theme danger
  border danger
}

text MutedLabel {
  color muted
}
```
